# Production Planning Automation using Linear Programming with VBA  

--- This project focuses on **optimizing production scheduling** in an agro-industrial company by automating the planning process and **reducing changeover time** using an **Excel VBA application**. The solution enhances operational efficiency by ensuring **optimal sequencing of production orders**, minimizing machine downtime, and improving overall productivity. 

## **Project Overview**

### **Problem Statement**
In the agro-food industry, particularly in tomato packaging, frequent series changeovers lead to significant production downtime. The goal of this project was to:
- **Minimize series changeover time** by optimizing the planning process.
- **Automate the generation of production plans** using Excel VBA.
- **Improve key performance indicators (KPIs)** such as service rate and changeover rate.

### **Solution**
- Developed an **Excel VBA application** to automate the planning process.
- Implemented **linear programming** techniques to optimize resource allocation and the sequencing of production.
- Reduced manual planning time by **90%** and improved the changeover rate by **20%**.

---

## **Key Features**
- **Automated Planning**: The VBA application generates optimized production plans in under 5 minutes.
- **Changeover Reduction**: Minimized the number of series changeovers by batching similar programs.
- **User-Friendly Interface**: Buttons and color-coded cells make the application easy to use.
- **Scalable Solution**: The application can be adapted to other production lines or industries.
